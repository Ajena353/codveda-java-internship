# ▶️ How to Run

### Clone the repository

```bash
git clone https://github.com/Ajena353/codveda-java-internship.git
```

### Navigate to the project

```bash
cd codveda-java-internship
```

### Compile

```bash
javac FileName.java
```

### Run

```bash
java FileName
```

Example:

```bash
javac Calculator.java
java Calculator
```
